**Nisaptham** is a minimalist, soul-focused music player designed for those who seek more than just sound. Built with React Native (Expo).

## Features
- **Local Library**: Scans device storage for MP3s.
- **Premium UI**: Clean, white/indigo aesthetic powered by NativeWind (Tailwind CSS).
- **Gestures**: Swipe-to-Queue, Drag-to-Dismiss Player.
- **Audio Engine**: Robust playback with background support.
- **Offline First**: All data stored locally in SQLite.

## Getting Started

### Prerequisites
- Node.js installed.
- Expo Go app on your Android/iOS device.

### Installation
1.  Clone the repo:
    ```bash
    git clone https://github.com/SNKR-23/Nisaptham.git
    cd Nisaptham/music-player
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App
1.  Start the Expo development server:
    ```bash
    npx expo start --clear
    ```
2.  **Scan the QR Code**:
    - **Android**: Open Expo Go app -> Scan QR code from terminal.
    - **iOS**: Open Camera app -> Scan QR code.

## Verification Checklist
- [ ] Scan Library: Ensure songs appear on first launch.
- [ ] Playback: Tap a song to verify audio.
- [ ] Queue: Swipe RIGHT on a song to add to queue.
- [ ] Player: Tap mini-player to expand full screen.
